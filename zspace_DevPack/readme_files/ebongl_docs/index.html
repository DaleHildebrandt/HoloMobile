EbonGL Graphics Library



EbonGL is an OpenGL library that lets us create stereo-vision holographic scenes with VRJuggler. It also runs on non-stereo systems with GLUT. It supports lighting, shadows, transparency and smooth shading. We can load 3D models with texture data, or draw 3D primitives directly. EbonGL is meant to greatly simplify the task of developing in 3D environments.
