Class EG_PointList



A list of OpenGL points. These points are drawn as dots in the scene. Each point has its own position, and the position of all points can be changed at once with setPosition.

Points are drawn as squares, or circles if antialiasing is enabled. They always face the viewer, and are not perspective-corrected. (Distant points will be drawn at the same size as nearby ones.) EG_PointList inherits from EG_Object.
