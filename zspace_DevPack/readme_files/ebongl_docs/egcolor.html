Class EG_Color



A 32-bit color. It contains a red, green and blue component to set the color, and an alpha value to set transparency.

  • EG_Color(GLfloat red, GLfloat green, GLfloat blue, GLfloat alpha)

  • void setColor(const EG_Color &newColor)
  • GLfloat* toArray(void)

EG_Color(void)

Default constructor, creates a color with red, green, blue and alpha all set to 1.


EG_Color(GLfloat red, GLfloat green, GLfloat blue, GLfloat alpha)

Constructor, creates the specified color.


EG_Color(const EG_Color &source)

Copy constructor, creates a color that's a copy of source.


EG_Color& operator= (const EG_Color &source)

Sets this color to source.


GLfloat alpha(void) const

Returns the color's alpha component. It ranges from 1 (completely solid) to 0 (completely transparent).


GLfloat blue(void) const

Returns the amount of blue in this color. The value ranges from 1 (max blue) to 0 (no blue).


GLfloat green(void) const

Returns the amount of green in this color. The value ranges from 1 (max green) to 0 (no green).


void setColor(GLfloat red, GLfloat green, GLfloat blue, GLfloat alpha)

void setColor(const EG_Color &newColor)

Sets this color to the specified values.


GLfloat* toArray(void)

Returns this color as an array of four values. In order, these are red, green, blue and alpha.


GLfloat red(void) const

Returns the amount of red in this color. The value ranges from 1 (max red) to 0 (no red).
