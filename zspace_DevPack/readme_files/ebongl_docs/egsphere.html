Class EG_Sphere



A 3D OpenGL Sphere. EG_Sphere inherits from EG_Object. The number of polygons making up a sphere is determined by the number of stacks and slices it has.

EG_Sphere(void)

Default constructor, creates a sphere with the slices and stacks set to 20, and radius set to 0.25.


EG_Sphere(GLdouble radius, GLdouble height, GLint slices, GLint stacks)

Normal constructor where you specify slices and stacks, and radius.


EG_Sphere(const EG_Sphere &source)

Copy constructor, creates a sphere that's a copy of source.


EG_Sphere& operator= (const EG_Sphere &source)

Sets this sphere to be a copy of source.


virtual EG_Object* generateInstance(void)

Returns a deep copy of this Sphere.


GLdouble getRadius(void)

Returns the sphere's radius.


GLint getSlices(void)

Returns the number of slices in the sphere. Slices refer to the number of subdivisions around the z-axis (similar to lines of longitude).


GLint getStacks(void)

Returns the number of stacks in the sphere. Stacks refer to the number of subdivisions along the z-axis (similar to lines of latitude)


void setRadius(GLdouble newRadius)

Sets the sphere's new radius.


void setSlices(GLint newSlices)

Sets the number of slices in the sphere.


void setStacks(GLint newStacks)

Sets the number of stacks in the sphere.
