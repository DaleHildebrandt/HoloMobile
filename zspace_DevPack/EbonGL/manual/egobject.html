Class EG_Object



All graphical objects that can be drawn in EG_Engine inherit from EG_Object. EG_Object is an abstract class that represents an object's position, rotation, color, and other graphical settings.

EG_Object(void)

Default constructor, creates a new graphics object.


EG_Object(const EG_Object &source)

Copy constructor, creates an EG_Object that's a copy of source.


EG_Object& operator= (const EG_Object &source)

Sets this object to a copy of source.


virtual EG_Object* generateInstance(void) = 0

This function should be implemented by a child class. It returns an exact copy of this object, and is used internally by EG_Engine.


EG_Color getAmbientDiffuse(void)

Returns this object's color.


virtual const EG_Polygon* getPolygon(unsigned int index) = 0

Returns one of the polygons making up this object. index indicates which polygon to retrive, and should be a number between 0 and numPolygons - 1.


MHTypes::Point3D getPosition(void)

Returns this object's center coordinates.


MHTypes::EulerAngle getRotation(void)

Returns this object's rotation, in degrees.


void hide(void)

Hides this object. The object is still part of the scene, but is not being drawn.


virtual unsigned int numPolygons(void) = 0

Returns the number of polygons in this object. Some objects, like points, might not have any polygons.


void setAmbientDiffuse(const EG_Color &newColor)

Set this object's color.


void setLighting(bool enabled)

If enabled is true, this object will be affected by light sources. If enabled is false, it will always be drawn at maximum brightness.


void setPolygonMode(GLenum newMode)

Set the mode for drawing polygon faces. newMode can take one of the following values: GL_POINT, GL_LINE or GL_FILL.


void setPosition(const MHTypes::Point3D &newPosition)

Set the position of the object's center, in scene coordinates.


void setShadowCasting(bool enabled)

If enabled is true, this object will cast shadows. Set enabled to false to disable shadows. Shadows will only be cast if the EG_Engine has shadows enabled.


void setShadowReceiving(bool enabled)

If enabled is true, this object will reflect shadows that were cast by another object.


void setRotation(const MHTypes::EulerAngle &newRotation)

Set the object's rotation. newRotation should be in degrees.


void show(void)

Start drawing this object in the scene.
