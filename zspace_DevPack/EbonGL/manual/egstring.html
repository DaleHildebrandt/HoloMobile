Class EG_String



EG_String is used when text needs to be sent between EbonGL and the user's program. We use our own custom string class rather than std::string, since std::string can't be exported from a DLL.

EG_String(void)

Default constructor, creates an empty string.


EG_String(const char *source)

Copy constructor, creates a string that's a copy of source. source should be a NULL-terminated C string.


EG_String(const EG_String &source)

Copy constructor, creates a string that's a copy of source.


EG_String& operator= (const EG_String &source)

Equal operator, sets this EG_String to be a copy of source.


void append(char newValue)

Appends a single character to the end of the string.


void appendCString(const char *source)

Appends a NULL-terminated C string to the end of the EG_String.


void clear(void)

Clear the string contents, setting size to zero.


static EG_String fromDouble(double source)

Creates an EG_String that contains the text version of source.


static EG_String fromInt(long int source)

Creates an EG_String that contains the text version of source.


char getChar(unsigned int index) const

Returns the character at the specified index in the string.


void setChar(unsigned int index, char newValue)

Set the character at the specified index to newValue.


unsigned int size(void) const

Returns the number of characters in the string.


const char* toCString(void) const

Returns this string as a NULL-terminated array of characters. The return value points to the EG_String's internal storage, and should not be modified. The values will change if the EG_String is later modified.


bool toInt(int &theInt) const

bool toInt(long int &theInt) const

Sets theInt to the integer contained within this string. Returns false if the string doesn't contain a valid integer.
