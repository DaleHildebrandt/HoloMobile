Class EG_TimeKeeper



Provides the current time at a millisecond resolution. Each EG_TimeKeeper stores a time value that can be used to determine how much time has elapsed: 

EG_TimeKeeper myTime;

//Store the current time
myTime.updateTime();

//Sleep for a second
sleep(1);

//Output the difference between the stored and current time
cout << myTime.timeElapsed() << endl;

EG_TimeKeeper(void)

Default constructor, creates a new EG_TimeKeeper with a stored time of zero.


EG_TimeKeeper(const EG_TimeKeeper &source)

Copy constructor, creates a EG_TimeKeeper with the same stored time as source.


EG_TimeKeeper& operator= (const EG_TimeKeeper &source)

Sets this object's stored time to be the same as source.


void addTime(uint32_t newTime)

Adds the specified number of milliseconds to the stored time.


uint16_t getMilliseconds(void)

Returns the milliseconds component of the stored time.


uint32_t getSeconds(void)

Returns the seconds component of the stored time.


void setTime(uint32_t seconds, uint16_t milliseconds)

Sets the stored time to the specified number of seconds and milliseconds.


uint32_t timeElapsed(void)

Returns the difference between the stored time and the current time.


void updateTime(void)

Update the stored time to the current time.
