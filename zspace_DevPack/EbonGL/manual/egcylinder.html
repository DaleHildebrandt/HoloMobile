Class EG_Cylinder



A hollow Cylinder. EG_Cylinder inherits from EG_Object. We can adjust the radius of the top and base seperately to create cone-like shapes. The number of polygons making up the cylinder is determined by its number of stacks and slices.

EG_Cylinder(void)

Default constructor, creates a cylinder with base and top set to 0.5, slices and stacks set to 20, and height set to 5.


EG_Cylinder(GLdouble radius, GLdouble height, GLint slices, GLint stacks)

Normal constructor where you specify slices and stacks, and height. The base and top equal to the radius given.


EG_Cylinder(GLdouble base, GLdouble top, GLdouble height, GLint slices, GLint stacks)

Glu-like constructor where you specify the base, top, height, slices and stacks.


EG_Cylinder(const EG_Cylinder &source)

Copy constructor, creates a cylinder that's a copy of source.


EG_Cylinder& operator= (const EG_Cylinder &source)

Sets this cylinder to be a copy of source.


virtual EG_Object* generateInstance(void)

Returns a deep copy of this Cylinder.


GLdouble getBase(void)

Returns the size of the Cylinder's base.


GLdouble getTop(void)

Returns the size of the Cylinder's top.


GLdouble getHeight(void)

Returns the Cylinders's height.


GLdouble getSlices(void)

Returns the number of slices in the Cylinder. Slices refer to the number of subdivisions around the z-axis (similar to lines of longitude).


GLdouble getStacks(void)

Returns the number of stacks in the Cylinder. Stacks refer to the number of subdivisions along the z-axis (similar to lines of latitude)


void setBase(GLdouble newBase)

Sets the Cylinder's new base size.


void setTop(GLdouble newTop)

Sets the Cylinder's new top size.


void setHeight(GLdouble newHeight)

Sets the Cylinder's new height.


void setSlices(GLint newSlices)

Sets the number of slices in the Cylinder.


void setStacks(GLint newStacks)

Sets the number of stacks in the Cylinder.
