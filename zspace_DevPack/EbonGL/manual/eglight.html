EG_Light(unsigned int newIndex)

This constructor is meant for use by EG_Engine, which creates a pool of lights for use by the program. Rather than creating your own lights, you should access the lights contained in EG_Engine.


EG_Light& operator= (const EG_Light &source)

Sets this light to have the same settings as source.


void disable(void)

Disable the light. Lights 1 to MAX_LIGHTS are disabled by default.


void enable(void)

Enable this light. It will begin illuminating the scene based on its position and color.


bool getEnabled(void)

Returns true if this light is currently illuminating the scene, false otherwise.


MHTypes::Point3D getPosition(void)

Returns this light's position. The position will only be valid for local lights.


void setAmbient(EG_Color newColor)

Set the light's ambient color. Ambient light:

  • Represents light that was reflected off one or more surfaces before illuminating the object.
  • Does not have an obvious source.
  • Scatters equally in all directions when it hits an object.


void setDiffuse(EG_Color newColor)

Set the light's diffuse color. Diffuse light:

  • Comes from one direction.
  • Is brighter if it hits the object straight on, rather than glancing off.
  • Scatters equally in all directions when it hits an object.


void setDirection(const MHTypes::EulerAngle &newAngle)

Set the light's direction to newAngle. This function currently has no effect on local lights, since spotlights aren't implemented. Local lights always emit light in all directions.


void setPosition(const MHTypes::Point3D &newPosition)

Set the light's position. This only has an effect for local lights.


void setToLocal(const MHTypes::Point3D &newPosition, const MHTypes::EulerAngle &newDirection)

Configure this EG_Light to be a local light. Local lights are affected by their position - for example, a light bulb is a local light. Currently, setting newDirection does nothing, since spotlights aren't yet implemented. Local lights always shine in all directions.


void setToDirectional(const MHTypes::EulerAngle &newDirection)

Configure this EG_Light to be a directional light. Directional lights have no position, and can be thought of as very far away. For example, the Sun is a directional light.


void setSpecular(EG_Color newColor)

Set the light's specular color. Specular light:

  • Comes from one direction.
  • Bounces off a surface in a preferred direction.
  • Adds highlights, makes objects appear shiny.


static const unsigned int MAX_LIGHTS

The number of OpenGL lights available in the scene. MAX_LIGHTS tells us how many lights can be accessed by EG_Engine's getLight function.
