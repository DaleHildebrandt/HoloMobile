Class EG_Utils



EG_Utils is a collection of utility functions that are used by multiple other classes within EbonGL. Creating an instance of EG_Utils and calling initEbonGL should always be the first step in any EbonGL program.

EG_Utils(void)

Default constructor, creates a new EG_Utils. There should only ever be one EG_Utils per program.


static void addLog(const EG_String &newEntry)

Adds newEntry to the current line to add to EbonGL's log file. When you're finished with the current line, use writeLog to output it to file and move to the next line.


void initEbonGL(unsigned int newViewports)

Initialize EbonGL. This function should be called before creating any graphics objects.


static void addVertex(std::vector &vertices, GLdouble vertexX, GLdouble vertexY, GLdouble vertexZ)

This convenience function adds a vertex to the specifed vector. We get three new elements, one for each of vertexX, vertexY, and vertexZ.


static void fatalError(void)

Report that a fatal error has occurred, and the program should halt. This is intended to catch internal bugs.


static std::string getErrMsg

Returns the most recent entry that was written to log file.


static unsigned int getNumViewports(void)

Returns the number of OpenGL viewports we're drawing to. Throws an exception if initEbonGL hasn't been called.


static MHTypes::Point3D getVertex(const std::vector &vertices, unsigned int index)

Returns the vertex from the specified index in vertices. If index is 4, we'll use elements 12, 13, and 14 to get the vertice's x, y and z coordinates.


static EG_LogWriter* log(void)

Returns a pointer to the EG_LogWriter EbonGL is using to record log messages.


static void setColor(std::vector &rgbaCol, MHTypes::MHFloat red, MHTypes::MHFloat green, MHTypes::MHFloat blue, MHTypes::MHFloat alpha)

Sets rgbaCol to be a four-element vector containing red, green, blue and alpha. Deletes any existing contents in rgbaCol.


static void writeLog(const EG_String &newEntry)

Outputs newEntry to the current line in EbonGL's log file, and moves to the next line.
