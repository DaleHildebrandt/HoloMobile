Class EG_Mutex



A simple cross-platform mutex.

  • EG_Mutex(void)
  • EG_String getErrMsg(void)
  • void lock(void)
  • void unlock(void)


EG_Mutex(void)

Default constructor. Throws an exception if we couldn't create a new mutex.


EG_String getErrMsg(void)

In the (very rare) event that one of the other classes throws an exception, you can get a description of the problem here.


void lock(void)

Lock this mutex, waiting until it becomes unlocked if necessary. Throws an exception if we couldn't lock it.


void unlock(void)

Unlock this mutex. Throws an exception if we run into an error.
