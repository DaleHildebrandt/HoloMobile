EG_Cuboid(void)

Default constructor, creates a cube with a width, height and depth of 1.


EG_Cuboid(const EG_Cuboid &source)

Copy constructor, creates a cuboid that's a copy of source.


EG_Cuboid& operator= (const EG_Cuboid &source)

Sets this cuboid to be a copy of source.


virtual EG_Object* generateInstance(void)

Returns a deep copy of this Cuboid.


GLdouble getDepth(void)

Returns the Cuboid's depth.


GLdouble getHeight(void)

Returns the Cuboid's height.


const EG_Polygon* getPolygon(unsigned int index)

Returns a polygon making up one of this Cuboid's faces. index should be a value between 0 and numPolygons.


MHTypes::Point3D getVertex(unsigned int index)

Returns a vertex making up one of this Cuboid's corners. index specifies the corner to retrieve, and should be a value between 0 and 7.


GLdouble getWidth(void)

Returns the Cuboid's width.


unsigned int numPolygons(void)

Returns the number of polygons making up this cuboid.


bool inside(const MHTypes::Point3D &thePoint)

Returns true if thePoint is inside of this Cuboid. This function only works properly if the Cuboid hasn't been rotated.


void setSize(GLdouble newWidth, GLdouble newHeight, GLdouble newDepth)

Set the Cuboid's width, height and depth.
