Human Computer Interaction Labs 3D Programming Suite



These tools and documentation are meant to greatly ease development on a 3D stereo-vision system that is configured with VRJuggler. They let us write programs on non-3D systems, and run the same program in stereo vision and VRJuggler's head-tracking.

EbonGL and HoloApp were developed by:

Human Computer Interaction Lab
Computer Science Department
University of Manitoba
Canada


Developer Tools Installation

This package contains all the libraries needed for development on a 2D system. To configure a new VRJuggler system, download the VRJuggler libraries from the HCI website. All libraries are 64-bit, and were compiled with Visual Studio 2010.

To install the libraries, copy the tools sub-directory to C:\. (root directory) You will need to add the C:\tools\allbin directory to your PATH variable. As of Windows 7, this can be done by:

  1. Start->Control Panel->System and Security->System
  2. Advanced System Settings->Environment Variables...
  3. Add C:\tools\allbin to the PATH System Variable. Note that paths are seperated by semicolons.

Afterwards, you will need to restart any terminal windows or instances of Visual Studio for the changes to take effect.


Compiling a Sample Program

The HoloLoader sample program is provided in the hololoader subdirectory. Creating a new Visual Studio project with VRJuggler support is very difficult. It's almost always easier to copy the hololoader solution and rename it with your program's name.

HoloLoader has two projects, both with the same source files. One project is for VRJuggler mode, and will produce an executable suitable for stereo 3D environments. The other is a GLUT project, which can run on almost any OpenGL capable computer.

With a few exceptions, the same code will work under both projects, letting us write code in GLUT, then re-compile it in VRJuggler mode when it's ready for the big screen.

MathHelper Library

MathHelper contains functions for rotating and translating 3D objects. It has quaternions, euler angles, 3D vectors, rotation matrices and more.

MathHelper Manual


EbonGL Library

The EbonGL library contains high-level features for displaying 3D objects. It can load models with textures, supports lighting and smooth shading, has basic shadowing and supports stereo vision through VRJuggler.

EbonGL Manual


HoloApp Class

The HoloApp class is included with the HoloLoader sample program. It lets us write programs that work with either GLUT or VRJuggler, and allows the same program to run on both stereo and non-stereo enabled systems.

HoloApp Manual
